share($property, [$property...,] $value)

Outputs multiple $property values with the same value. For example:

// Input:
.element {
  @include share(padding-top, padding-bottom, 20px);
}

// Output:
.element {
  padding-top: 20px;
  padding-bottom: 20px;
}

arrow($size: 10px, $color: inherit, $direction: down)

Makes a CSS arrow shape. $direction can be one of the following, and affects the direction the arrow points in:

  • down (default)
  • up
  • left
  • right
